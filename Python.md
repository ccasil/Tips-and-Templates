##### For Loop Template
```Python
fruits = ["apple", "banana", "cherry"]
for x in fruits:
    print(x)
    # loop through a list

for x in "persimmon":
    print(x)
    # loop through a string

for x in range(0, 10):
    print(x)
    # 0 to 9 (not including 10)
```
